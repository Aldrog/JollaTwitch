TwitchTube
==========

Twitch.tv client for Sailfish OS.

Features
--------
- Navigate by game
- View top channels
- Browse followed channels and games
- Follow and unfollow channels and games
- Search channels
- Chat

State of development
--------------------
Version 1.0 has been released and waiting for the Harbour QA.
